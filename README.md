### 기능 요구사항

- 자동차 
  - 자동차 이름
    - 이름 제한조건은 5자리. 그 이상은 초과불가
    - 각 이름은 , 로 구분한다.
    - 이름이 5자리가 넘어가면 예외를 발생하고 다시 값을 입력받는다. 
  - 자동차 전진
    - 생성된 난수가 4 이상이면 전진, 미만이면 전진 안한다. 
    - 전진 수만큼 '-' 표시
- 시도 횟수 입력
  - 시도 횟수 입력 시 숫자이외의 입력값은 예외를 발생시킨다.
- 출력 
  - 가장 전진 수 가 많은 우승자 출력
  - 우승자는 여러명 일 수 있다.
